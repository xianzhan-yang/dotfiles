## Install Basic Packages

```
sudo pacman -S alacritty feh picom rofi zathura polybar exa starship
```

## Use Configuration Manually

```
cp -r dotfiles/{i3,alacritty,picom,polybar,rofi,gtk-3.0,zathura} ~/.config
```

## Use Wallpaper

```
mkdir ~/Pictures

cp dotfiles/1.png ~/Pictures
```

## Use Bash Configuration

```
cp dotfiles/.bashrc ~/
```
