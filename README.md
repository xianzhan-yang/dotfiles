## Install Packages

```
sudo pacman -S alacritty feh picom rofi zathura polybar exa starship pulseaudio pulseaudio-alsa
```

```
paru -S nordic-theme
```

## Use it

```
cp -r dotfiles/{i3,alacritty,picom,polybar,rofi,gtk-3.0,zathura} ~/.config
```

```
mkdir ~/Pictures

cp dotfiles/1.png ~/Pictures
```

```
cp dotfiles/.bashrc ~/
```
